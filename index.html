  • 快速

    Node.js的为您带来难以置信的生成速度。数百个文件需要几秒钟即可完成。

  • Markdown支持

    支持所有GitHub中的Markdown功能，你甚至可以在Hexo中使用Octopress插件。

  • 一键部署

    你只需要一个命令就可以将你的网站部署在GitHub Pages、Heroku或其他网站。

  • 丰富插件

    Hexo具有强大的插件系统。您可以安装Jade、CoffeeScript等更多的插件。

  • npm install hexo -g
  • hexo init blog
  • cd blog
  • npm install
  • hexo server
